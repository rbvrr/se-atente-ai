# Se atente Aí
Um dispositivo IoT baseado na técnica Pomodoro, com sensores inteligentes que monitora, adaptam e otimizam sessões de foco — em qualquer contexto. Tecnologia nordestina para manter você presente.
- Projeto desenvolvido no Mestrado de Engenharia de Software 2026.1 (Cesar School)
- Disciplina: Interoperabilidade
